# OpenSquilla — Token-Efficient AI Agent

**TL;DR:** OpenSquilla is a Python agent framework that maximizes "intelligence density" — getting higher-quality AI output from fewer tokens. It compresses prompts, routes to registered skills, and persists memory across sessions, all within a declared token budget.

## Headline Result

```
Original prompt:  89 tokens
Compressed:       32 tokens  (64% savings)
Same task completed — 2.8x intelligence density improvement
```

## Quick Start

```bash
bash run.sh
```

No API keys needed — the demo runs entirely with mock data.
