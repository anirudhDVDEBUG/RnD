# Claude Pets — Desktop Pet for Claude Code

**Your coding sessions get a live desktop companion that reacts in real time.** Claude Pets connects Claude Code's lifecycle hooks and an MCP server to the [OpenPets](https://openpets.dev) desktop app, so a pet on your screen wakes up, thinks, celebrates, and sleeps alongside your AI-assisted coding.

```
  (o_-)~  Claude is using Agent: explore codebase  [thinking]
  (>_<)/*  Claude calls Edit: src/app.js            [writing]
  \(^o^)/  All changes applied — tests pass!        [success]
```

**Quick links:** [Setup & Install](HOW_TO_USE.md) | [Technical Details](TECH_DETAILS.md)

## Run the demo

```bash
bash run.sh
```

No API keys or external services needed — the demo runs in mock mode, simulating a full Claude Code session with pet state transitions.
