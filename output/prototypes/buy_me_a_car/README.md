# Buy Me A Car

Claude Code skill that runs end-to-end used car buying: multi-site listing search, CARFAX red-flag analysis, 50-state OTD price breakdowns, mass dealer outreach templates, negotiation anchoring from KBB/Edmunds/NADA, and a decision tracker that updates as deals progress. Say "help me find a used car under $25k" and Claude handles the rest.

**Headline result:** Five listings compared, CARFAX scored, OTD calculated to the penny for Texas, dealer emails drafted, and a final "buy this one" recommendation -- all from one prompt.
