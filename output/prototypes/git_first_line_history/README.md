# Git First Line History

**Track how the first line of any file has changed across Git history** — useful for catching project renames, title evolution, or version bumps at a glance.

## Headline Result

```
2025-11-24 16dfc1a  # Warelay — WhatsApp Relay CLI (Twilio)
2025-11-25 4d2a8a8  # warelay — Send, receive, and auto-reply on WhatsApp
2025-12-01 a3b2c1d  # CLAWDIS
2025-12-15 e4f5a6b  # OpenClaw
```

One command reveals the full naming history of a project — no manual log digging required.
