# AI Media Generator

Claude Code skill for high-quality AI image, video, and music prompt crafting and browser-based execution across 14+ generative platforms.

## When to use

- "Generate an AI image prompt for a cinematic sunset scene"
- "Create a video prompt for Kling / Sora / Veo / Seedance"
- "Write a Midjourney prompt for a product photo"
- "Craft a music prompt for Suno"
- "Help me generate media across multiple AI platforms"

## How to use

1. **Describe your creative vision**: Tell Claude what kind of media you want — image, video, or music — along with the mood, style, subject, and any specific details.

2. **Select target platform(s)**: Choose from 14+ supported generative AI platforms:
   - **Image**: Midjourney, Flux, DALL-E, Stable Diffusion, Ideogram
   - **Video**: Sora, Kling, Veo, Seedance, Runway, Pika
   - **Music**: Suno, Udio
   - Or let Claude recommend the best platform for your use case.

3. **Prompt generation**: Claude crafts a senior-director-level prompt optimized for the selected platform, incorporating:
   - Cinematic composition and framing techniques
   - Lighting, color grading, and atmosphere direction
   - Camera movement and lens specifications (for video)
   - Platform-specific syntax, parameters, and best practices
   - Negative prompts where applicable

4. **Iterate and refine**: Adjust the generated prompt by requesting changes to style, mood, aspect ratio, duration, or technical parameters.

5. **Browser-based execution**: When ready, Claude can open the target platform in your browser and submit the prompt directly for generation.

## Prompt Crafting Principles

- **Cinematic language**: Use professional film direction vocabulary — shot types, lens choices, lighting setups
- **Platform awareness**: Each platform has unique syntax, strengths, and parameter formats; prompts are tailored accordingly
- **Structured layering**: Build prompts in layers: subject, environment, lighting, camera, style, technical parameters
- **Negative prompting**: Include negative prompts for platforms that support them to avoid common artifacts
