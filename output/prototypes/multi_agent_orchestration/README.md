# Multi-Agent Orchestration Engine

**TL;DR:** A Python framework for orchestrating multiple AI agents in sequential pipelines, parallel fan-outs, or hierarchical delegation patterns. Zero dependencies, runs instantly, and maps directly to Claude Code's Agent tool for real swarm deployments.

```
    1. Sequential Pipeline  - 4 agents in series
    2. Parallel Fan-out     - 5 agents in parallel (2.8x speedup)
    3. Hierarchical         - 1 coordinator + 3 delegates
  Total agents invoked: 13
```
