# Unslop Text

**Strip AI-generated fluff from any text in one pass.** A zero-dependency Python library + Claude Code skill that detects and removes filler adverbs, hollow adjectives, cliched transitions, and sycophantic openers that scream "a chatbot wrote this."

### Headline result

```
Input  (181 words):  "Great question! In today's rapidly evolving digital
                      landscape, leveraging cutting-edge AI tools..."
Output (89 words):   "AI tools have become essential for businesses looking
                      to improve their content strategy..."
Slop removed: 38 instances — 44% shorter, 100% less cringe.
```
