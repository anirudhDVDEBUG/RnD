# Hypothesis Validator

**A Claude Code skill that stress-tests startup hypotheses before you write a line of code.** Structured framework covering ICP clarity, demand evidence, money signals, and go-to-market feasibility — outputs a scorecard with a green/yellow/red verdict.

## Headline Result

```
| Dimension       | Rating       | Key Finding                                    |
|-----------------|--------------|------------------------------------------------|
| ICP Clarity     | Sharp        | Named 5+ prospects, active in IH & Twitter/X   |
| Evidence        | Supporting   | Google Trends +340% YoY for "AI meeting notes"  |
| Money Signals   | Strong       | Competitors charging $10-30/mo, 1M+ users       |
| GTM Feasibility | Clear path   | Self-serve, organic channels, solo-buildable     |

Overall Verdict: [GREEN] Validate further
```
