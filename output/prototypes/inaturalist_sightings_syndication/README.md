# iNaturalist Sightings Syndication

**TL;DR:** Pull your iNaturalist wildlife observations into a static site as first-class content — with photo grids, species names, pagination, and an Atom feed — using a single Python script.

## Headline Result

```
$ bash run.sh
Synced 12 sightings (47 species) -> output/
Generated: index.html, page1.html, page2.html, feed.xml
Open output/index.html to browse your sightings gallery.
```
