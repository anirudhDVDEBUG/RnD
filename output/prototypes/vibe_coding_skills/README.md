# Vibe Coding Skills

**A Claude skill that structures "vibe coding" — describe what you want in plain English and let AI agents build it, with a proven 4-step methodology that turns vague requests into reliable code.**

## Headline Result

```
Input:  "I need a search bar in React with TypeScript that filters products
         by name with debounce. Handle loading and no-results states."

Output: A structured 4-step plan: context grounding, intent analysis
        (UI elements, behaviors, edge cases detected), iteration sequence,
        and a validation checklist — all before a single line of code.
```

The skill teaches Claude to guide you through **Ground > Describe > Iterate > Validate** instead of dumping code on the first prompt.

## Quick Start

```bash
bash run.sh
```

No API keys needed. Produces a structured vibe-coding plan from example requests.
