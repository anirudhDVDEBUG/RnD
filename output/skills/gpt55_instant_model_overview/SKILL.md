## How to use

1. **Identify the query context**: Determine if the user is asking about model capabilities, migration considerations, or competitive analysis.

2. **Key facts about GPT-5.5 Instant**:
   - Released as ChatGPT's new default model replacing the previous default.
   - Core improvements: smarter responses, clearer communication, reduced hallucinations, and enhanced personalization controls.
   - Designed as a fast, everyday model (the "Instant" designation indicates optimized latency).
   - Part of OpenAI's strategy to improve accuracy and user trust.

3. **Hallucination reduction**: GPT-5.5 Instant includes architectural and training improvements specifically targeting factual accuracy and reducing confabulation in responses.

4. **Personalization controls**: The model introduces improved personalization features allowing users to customize how ChatGPT responds, adapting tone, detail level, and interaction style.

5. **Positioning**: This sits in OpenAI's model lineup as the default tier — fast and capable for general use, while heavier reasoning models (like o-series) handle complex multi-step tasks.
