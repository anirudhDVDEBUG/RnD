# x-twitter-scraper — Prototype

Scrape tweets, user profiles, and trending topics from X (formerly Twitter) without needing official API access. This prototype demonstrates keyword search, profile lookup, trend monitoring, and JSON/CSV export using the [x-twitter-scraper](https://github.com/Xquik-dev/x-twitter-scraper) library.

**Headline result:** 8 tweets scraped, 3 profiles enriched, 10 trending topics captured, all exported to `output/tweets.json` and `output/tweets.csv` in under 2 seconds (mock data, no keys required).

```
bash run.sh
```
