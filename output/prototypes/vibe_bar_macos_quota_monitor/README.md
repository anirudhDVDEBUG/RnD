# Vibe Bar — macOS Quota Monitor

**TL;DR:** Vibe Bar is a native macOS SwiftUI menu-bar app that shows Claude Code and Codex subscription quota consumption, usage pace, token cost, and service status at a glance. This repo contains a terminal demo that simulates the same dashboard with mock data — no API keys needed.

## Headline Result

```
  Quota used:  $94.17 / $200.00  (47.1%)
  [##############................]
  Pace:        =  ON PACE   (good)
  Projected:   $188.34 by end of period
```

One look tells you whether to keep coding or ease off for the rest of the billing period.

## Quick Start

bash run.sh
