# QR Code Generator

A single-file, zero-dependency HTML tool that generates QR codes for URLs, plain text, and WiFi network credentials entirely in the browser. No server, no API keys, no build step — just open the file.

**Headline result:** Paste a WiFi SSID + password, click Generate, and get a scannable QR code PNG that guests can point their phone camera at to connect instantly.
