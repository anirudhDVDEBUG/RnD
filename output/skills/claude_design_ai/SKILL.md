# Claude Design AI

AI-powered UI/UX architect skill for Claude Code. Converts visual designs into production-ready React components with Tailwind CSS.

## When to use

- "Convert this screenshot to a React component"
- "Generate a Tailwind CSS component from this design"
- "Create a design system with dark mode support"
- "Build a responsive layout from this wireframe"
- "Create an SVG icon set for my project"

## How to use

1. **Screenshot to React**: Provide a screenshot or describe a UI, and this skill will generate a complete React component with Tailwind CSS styling.

2. **Design System Generation**: Request a design system and receive a complete set of reusable components including:
   - Color tokens and typography scales
   - Button, input, card, and modal components
   - Dark mode toggle and theme provider
   - Consistent spacing and layout primitives

3. **Wireframe to Code**: Describe or provide a wireframe layout and get:
   - Responsive React components using Tailwind CSS
   - Mobile-first breakpoint handling
   - Proper semantic HTML structure
   - Accessibility attributes (ARIA labels, roles)

4. **Component Export**: Generate production-ready components with:
   - shadcn/ui integration when available
   - TypeScript props interfaces
   - Tailwind CSS utility classes
   - SVG icon components and vector assets

5. **Branding & Theming**: Create branded component libraries with:
   - Custom color palettes from brand guidelines
   - Consistent typography and spacing
   - Dark/light mode theme switching
   - CSS custom properties for theme tokens

### Workflow

```
1. Analyze the design input (screenshot, wireframe description, or Figma reference)
2. Identify components, layout structure, and design tokens
3. Generate React components with Tailwind CSS
4. Add responsive breakpoints and dark mode support
5. Export clean, production-ready TypeScript code
```

### Key capabilities

- **Screenshot → React**: Analyze visual designs and produce matching React/Tailwind components
- **Figma components**: Translate Figma design specs into coded components
- **Tailwind CSS generator**: Generate utility-first CSS with proper responsive design
- **Prototyping agent**: Rapidly scaffold interactive UI prototypes
- **SVG icon creator**: Generate and optimize SVG icons as React components
- **Dark mode toggle**: Implement theme switching with CSS variables and Tailwind dark: variants
- **Responsive layouts**: Build mobile-first layouts with Tailwind breakpoints
- **shadcn/ui integration**: Generate components compatible with the shadcn/ui library
