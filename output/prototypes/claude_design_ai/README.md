# Claude Design AI

**AI-powered UI/UX architect skill for Claude Code.** Say "create a design system" or "convert this screenshot to React" and get production-ready TypeScript components with Tailwind CSS, dark mode, responsive layouts, and SVG icons — all generated in seconds.

## Headline result

Running `bash run.sh` generates **8 React components + config** in a complete design system: ThemeProvider, DarkModeToggle, Button (5 variants), Card, Input, Navbar (mobile-responsive), Hero section, and an SVG icon library — plus a fully-assembled landing page using all of them.
