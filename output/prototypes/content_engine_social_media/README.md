# Content Engine - Social Media

**One topic in, five platform-ready posts + AI image out.** This Claude Code skill generates channel-adapted content for Twitter/X, LinkedIn, Instagram, Facebook, and Blog/Newsletter from a single topic, creates a matching AI image via fal-ai, and auto-publishes via Upload-Post -- all in one conversation, zero n8n nodes.

## Headline Result

```
$ bash run.sh
[1/3] Generating content for 5 channels (mock mode)...
  --- Twitter / X (167/280 chars) ---
  --- LinkedIn (598/1300 chars) ---
  --- Instagram (412/2200 chars) ---
  --- Facebook (318/500 chars) ---
  --- Blog / Newsletter (742/3000 chars) ---
[2/3] Generating image (mock SVG)...
[3/3] Publishing to channels (mock mode)...
  Pipeline complete! 5 posts + 1 image saved.
```
