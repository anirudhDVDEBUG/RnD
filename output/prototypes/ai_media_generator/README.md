# AI Media Prompt Generator

**Zero-skill cinema.** A Claude Code Skill that crafts senior-director-level prompts for AI image, video, and music generation, then executes them via browser across 14+ generative platforms (Midjourney, Sora, Kling, Veo, Flux, Suno, and more).

        grading, shot on 85mm portrait lens, premium aspirational clean

One sentence in, platform-optimized cinematic prompt out.
