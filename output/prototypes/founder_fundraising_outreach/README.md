# Founder Fundraising Outreach

A Claude skill that generates investor outreach emails founders actually send — cold emails, follow-ups, warm intro requests, and due diligence prep docs. No API keys needed; structured templates enforce the brevity and specificity that get replies.

**Headline result:** From a company profile + investor target, produces four ready-to-send artifacts in under a second — each following the "5-8 sentences, one CTA, metrics over adjectives" formula that investors respond to.

- [TECH_DETAILS.md](TECH_DETAILS.md) — Architecture, data flow, limitations
