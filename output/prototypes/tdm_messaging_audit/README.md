# TDM Messaging Audit

**Audit and rewrite product messaging so Technical Decision Makers actually buy your thing.** Most B2B buyers (the "90% TDM") don't browse Hacker News — they read Gartner reports and need a defensible purchasing decision. This Claude Code skill scores your copy on 5 dimensions and rewrites the weak parts.

## Headline Result

```
  AcmeDB (Before)           — TDM Resonance: 1.4 / 5.0  ← "blazingly fast" won't survive procurement
  AcmeDB (After TDM Rewrite) — TDM Resonance: 5.0 / 5.0  ← analyst-proof, career-safe, defensible
```
