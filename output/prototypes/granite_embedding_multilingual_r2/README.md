# Granite Embedding Multilingual R2

IBM's open-source (Apache 2.0) multilingual embedding models: 97M and 311M parameter variants built on ModernBERT, supporting 200+ languages with 32K token context. The 97M model is the best sub-100M retrieval model on MTEB Multilingual, scoring 60.3 vs 50.9 for multilingual-e5-small.

**Headline result:** Cross-lingual retrieval works out of the box — query in English, retrieve documents in Japanese, German, or Spanish with no language-specific configuration.
