# AI Engineering Skill Pack for Claude Code

A methodology-first Claude Code skill that gives you structured workflows for the five tasks every AI engineer repeats: RAG evaluation, agent debugging, MCP server scaffolding, paper reproduction, and benchmark reporting. Say "evaluate my RAG pipeline" and Claude walks through retrieval metrics, generation quality, and actionable recommendations — no separate eval framework required.

**Headline result:** Run `bash run.sh` to see three complete evaluation reports (RAG, agent debug, benchmarks) generated from mock data in under 2 seconds with zero dependencies.
