# Cybrix Deploy - Claude Code Skill for Auto-Deployment

**TL;DR:** A Claude Code skill that lets you deploy and manage hosted applications on the Cybrix platform using natural language commands. Say "deploy this app to Cybrix" and it handles framework detection, build config, and auto-deployment from Git.

## Headline Result

```
You: "Deploy this app to Cybrix"
Claude: Detected Next.js project. Building... Deployed to https://myapp.cybrix.app
        Auto-deploy enabled on push to main.
```
