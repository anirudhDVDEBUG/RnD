# Parloa Voice Service Agent

**A working prototype of a Parloa-style voice AI customer service agent** — dialogue flows, mock STT/TTS, backend tool calling, and a simulation harness that runs 5 test scenarios with zero API keys.

> **Headline result:** 5 conversation scenarios (billing, tech support, escalation, account inquiry, adversarial) execute end-to-end in under 1 second with sub-300ms simulated per-turn latency, tool invocations verified, and guardrails tested.
