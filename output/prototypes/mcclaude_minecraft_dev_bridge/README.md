# McClaude — Minecraft Dev Bridge

**Develop Minecraft plugins from Claude Code on a live server.** McClaude is an MCP bridge that lets you read the server console, run commands, edit Skript files, and eval code — all from your IDE with end-to-end encryption.

**Headline result:** Edit a Skript file, reload, and see the change live on a running Minecraft server — without ever leaving Claude Code.
