# MetalBear Skills Registry - Claude Code Skill

**Browse, search, and install reusable Claude Code skills from the MetalBear community registry — without leaving your editor.**

## Headline Result

```
$ python skills_registry.py search "kubernetes"
Found 3 skills matching "kubernetes":
  k8s-debug        - Debug Kubernetes pods, services, and deployments interactively
  helm-assist      - Generate and validate Helm charts with Claude
  k8s-migration    - Migrate workloads between clusters with AI guidance

$ python skills_registry.py install k8s-debug
Installed k8s-debug -> .claude/skills/k8s-debug.md
```
