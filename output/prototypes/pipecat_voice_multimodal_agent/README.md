# Pipecat Voice & Multimodal Agent

**Build real-time voice AI agents with swappable STT/LLM/TTS in under 50 lines.** Pipecat is a 14k-star open-source framework gaining +101 stars/day that turns conversational AI into composable pipelines — plug in Deepgram, OpenAI, Cartesia, or any provider and ship a voice bot over WebRTC, WebSocket, or phone.

**Headline result:** This demo runs a 5-turn mock voice conversation through the full Pipecat pipeline architecture (VAD -> STT -> LLM -> TTS) with frame tracing, zero API keys required.

```
bash run.sh
```
