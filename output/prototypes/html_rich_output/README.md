# html_rich_output — Claude Code Skill

**Tell Claude to output rich, interactive HTML instead of flat Markdown.** One SKILL.md file gives you color-coded PR reviews with inline SVG diagrams, collapsible sections, filterable findings, and syntax-highlighted code — all in a single self-contained `.html` file you open in any browser.

**Headline result:** A 6-finding PR review rendered as a polished, interactive HTML page with severity filtering, architecture diagram, and inline code suggestions — generated in under a second with zero dependencies.
