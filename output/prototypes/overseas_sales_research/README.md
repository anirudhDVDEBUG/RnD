# Overseas Sales Research Skill (五看六定)

**A Claude Code skill that generates structured B2B overseas sales research reports in ~30 minutes using the "Five Perspectives, Six Decisions" framework** — covering industry landscape, competitive analysis, customer profiling, and a phased 30/60/90-day go-to-market plan.

> **Headline result:** Ask Claude "Research Siemens AG for a distributor evaluation in Southeast Asia" and get a complete, stakeholder-ready report with market sizing, competitor benchmarking, SWOT analysis, and an actionable sales playbook — all in a single Markdown file.
