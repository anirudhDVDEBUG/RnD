# GPT-5.5 Instant System Card Analyzer

**TL;DR:** Parses and summarizes OpenAI's GPT-5.5 Instant System Card into a structured safety report covering Preparedness Framework risk ratings, benchmark comparisons (vs GPT-4o and GPT-5), red teaming findings, and deployment mitigations -- all runnable with zero dependencies.

**Headline result:** GPT-5.5 Instant scores **LOW risk across all four Preparedness Framework categories** (CBRN, cybersecurity, persuasion, model autonomy) while improving on GPT-4o safety benchmarks in 5/5 categories.

## Quick start

```bash
bash run.sh
```
