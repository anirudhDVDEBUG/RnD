# Social Media Scraper Skill

**Drop a social media link into Claude Code, get everything back:** metadata, comments, transcription, and AI video analysis. Supports Instagram, TikTok, Twitter/X, and YouTube.

## Headline Result

```
You:    "Scrape this YouTube video: https://youtube.com/watch?v=abc123"
Claude: ### Post Metadata
        - Author: @TechFounderAI
        - Views: 284,000 | Likes: 12,400 | Comments: 873
        ### Transcription
        "So I've been running this AI agent for about six weeks now..."
        ### Top Comments
        1. @agent_builder_99: This is exactly what I've been trying to build (142 likes)
```

One command. No manual copy-paste. Structured output ready for analysis.
