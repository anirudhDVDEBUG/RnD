Y Combinator owns ~0.6% of OpenAI — worth over **$5 billion** at the $852B valuation (May 2026). This Claude Code skill and standalone demo lets you instantly surface those numbers, compare them against other legendary accelerator returns, and run sensitivity analysis at different valuations.

```
YC's ~$150K investment → $5.1B stake value → ~34,000x return
The single largest dollar-value accelerator return in history.
```
