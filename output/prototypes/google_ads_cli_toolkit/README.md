# Google Ads CLI Toolkit

**Manage Google Ads campaigns, GTM containers, and GA4 analytics entirely from the terminal.** This open-source toolkit by [@nicolasmaldonadoj](https://github.com/nicolasmaldonadoj) ships with a Claude Code playbook so you can compose GAQL queries, pull performance reports, and automate bulk ad operations through natural language.
