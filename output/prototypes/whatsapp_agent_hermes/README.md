# WhatsApp Agent for Hermes

**Turn Hermes into an autonomous WhatsApp manager.** Schedule messages, send voice notes, share files from your PC, and search contacts -- all hands-free via Andorina.

```
bash run.sh
```

**Headline result:** A single `agent.schedule_message("+1-555-0103", "Good morning!", delay_seconds=60)` call queues, waits, and delivers the message through WhatsApp Web -- no manual typing required.
