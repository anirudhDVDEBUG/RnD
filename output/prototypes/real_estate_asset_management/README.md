# Real Estate Asset Management — Claude Code Skill

**TL;DR:** A Claude Code skill that automates quarterly asset reviews, LP reporting, rent roll analysis, T-12 operating statements, valuations, debt monitoring, and hold/sell/refi decisions for commercial real estate portfolios.

## Headline Result

```
RECOMMENDED ACTIONS
  Riverfront Office Tower       -> Hold   | Lease-up/value-add runway remains
  Sunset Gardens Multifamily    -> Sell   | Stabilized with $6.9M unrealized gain
  Metro Industrial Park         -> Refi   | Debt matures in 8 months; refinance to extend
```

Run `bash run.sh` to see a full Quarterly Asset Review across a 3-property portfolio with T-12 analysis, WALT calculations, debt covenant flags, and disposition scoring — all from mock data, no API keys needed.
