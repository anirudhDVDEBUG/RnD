# Token Cost Lint

**Static token waste auditor for multi-agent Claude Code harnesses.** Classifies bloat across 10 categories and 31 sub-patterns using zero LLM calls — the linter itself costs nothing to run.

**Headline result:** Run `bash run.sh` on the included sample project and see ~20%+ estimated token waste identified in under 1 second, with prioritized fix recommendations.
