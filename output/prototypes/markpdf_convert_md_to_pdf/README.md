# MarkPDF — Markdown to Premium PDFs, Slides & HTML

**Convert any `.md` file into a professionally styled PDF, slide deck, or HTML page — directly from Claude Code.** MarkPDF is a Claude Skill + MCP server that triggers whenever you say "convert this markdown to PDF" or "generate slides from my notes." Zero config, premium output.

**Headline result:** A quarterly report markdown file becomes a print-ready, syntax-highlighted HTML document (and slide deck) in under 2 seconds — open in browser and print to PDF.

## Quick Start

```bash
bash run.sh
# → output/sample.html        (styled document)
# → output/sample_slides.html (slide deck)
```
