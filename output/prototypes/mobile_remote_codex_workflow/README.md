# Mobile & Remote Codex Workflow Manager

**Manage, monitor, steer, and approve AI-assisted coding tasks in real time from any device — including your phone.**

OpenAI's Codex (available via the ChatGPT app) lets you start coding tasks from desktop and approve them from mobile. This prototype demonstrates the full lifecycle: task creation, real-time progress monitoring, mid-task steering, diff review, and one-tap approval — all from a simulated mobile device.

## Headline Result

```
[APPROVED] Add input validation to the signup form (+61/-4 across 3 files)
  Reviewed and approved from: iOS (ChatGPT App)
  PR Created: https://github.com/myorg/webapp-frontend/pull/427
```
