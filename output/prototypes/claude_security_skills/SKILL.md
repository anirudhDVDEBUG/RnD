# Claude Security Skills

25 production-tested defensive security skills for Claude Code. When triggered, systematically audit the user's codebase, configurations, and infrastructure against security best practices.

## Skill Categories

### Infrastructure: VPS, Cloudflare, Docker, GitHub Actions
### Web Apps: WordPress, Next.js, Webshell Detection
### AI/LLM: Prompt Injection, OWASP LLM Top 10, MCP Security, Agent Guardrails, Slopsquatting, Hallucinated APIs, Sycophancy
### Response: Incident Response, Log Analysis, Forensics
### Compliance: GDPR, DACH

## Audit Methodology

1. Identify the user's stack from project files
2. Run the matching checklists systematically
3. Report findings with severity (Critical/High/Medium/Low)
4. Provide concrete remediation with code/config changes
5. Reference standards (OWASP, CIS, NIST) where applicable
