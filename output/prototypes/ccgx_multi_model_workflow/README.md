# CCGX Multi-Model Workflow

**Auto-route dev tasks to the right AI model.** Frontend work goes to Gemini, backend work goes to Codex, and Claude orchestrates — with configurable quality tiers (draft / standard / production) and autonomous long-run support.

**Headline result:** 8 mixed tasks classified and routed to 3 models in <2 seconds, with per-task confidence scores and token/latency tracking.
