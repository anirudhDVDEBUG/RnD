# Finance Codex Reporting

**TL;DR:** A Claude Code skill that generates finance team deliverables — MBRs, variance bridges, model checks, and planning scenarios — from structured data, using the same workflows OpenAI's Codex Academy recommends for finance teams.

## Headline Result

```
VARIANCE BRIDGE — FY26 Q1 Budget → Actual
  Starting point: Budget Net Income = $1,020K
  + Product Revenue:     +$600K  (launch ahead of schedule)
  + Service Revenue:     +$200K  (consulting upsells)
  ...
  = Actual Net Income:   $1,232K  (+$212K / +20.8%)
  Reconciliation check: PASS
```

One command produces four complete finance deliverables from mock data — no API keys required.

## Quick Start

```bash
bash run.sh
```
