# CosmoBlk Email Design — Prototype

**Anti-slop email design for Claude Code.** This skill generates production-ready MJML/HTML marketing emails using six committed archetypes (Welcome, Promotional, Newsletter, Transactional, Re-engagement, Announcement), brand-aware briefs, real imagery, and compliant footers. Works with any ESP via MCP.

## Headline result

Run `bash run.sh` → get six fully rendered, responsive HTML emails in `output/`, each with real copy, branded colors, proper CAN-SPAM footer, and ESP-specific merge tags — zero manual HTML editing.
