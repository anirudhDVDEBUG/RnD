# WhatsApp Agent for Hermes

Set up and configure the Andorina WhatsApp Agent for Hermes — an autonomous WhatsApp manager that lets you schedule messages, send voice notes, share PC files, and search contacts instantly.

## When to use

- "Set up a WhatsApp automation agent with Hermes"
- "I want to schedule WhatsApp messages automatically"
- "Help me configure Andorina WhatsApp bot"
- "Send voice notes and files via WhatsApp using AI"
- "Build an autonomous WhatsApp messaging agent"

## How to use

1. **Clone the repository**
   ```bash
   git clone https://github.com/AndorinaAI/Andorina-WhatsApp-Agent-for-Hermes.git
   cd Andorina-WhatsApp-Agent-for-Hermes
   ```

2. **Install dependencies** (Python 3 required)
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure the agent**
   - Set up your WhatsApp Web session (the agent interfaces with WhatsApp via browser automation)
   - Configure Google Contacts integration if you want contact search capabilities
   - Review and adjust any environment variables or config files in the project

4. **Key capabilities**
   - **Schedule messages**: Queue messages to be sent at specific times
   - **Send voice notes**: Generate and send voice notes through WhatsApp
   - **Share files**: Send files from your PC directly through WhatsApp
   - **Search contacts**: Instantly search and find contacts (Google Contacts integration)
   - **Autonomous management**: Runs as a Hermes agent skill for hands-free operation

5. **Run the agent**
   ```bash
   python main.py
   ```

6. **Integration with Hermes**
   - This is designed as a skill/plugin for the Hermes agent framework
   - Once installed, Hermes can autonomously manage your WhatsApp messaging
   - The agent runs on Linux with Python 3 and operates locally for privacy
