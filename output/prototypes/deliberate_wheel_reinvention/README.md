# Deliberate Wheel Reinvention

A Claude Code skill that turns "reinvent the wheel" from an insult into a learning strategy. Tell Claude what domain you want to master, and it picks the right 4-5 things to rebuild from scratch — scoped small enough to finish, deep enough to teach transferable principles.

**Headline result:** 18 curated reimplementation targets across 6 domains, each with a concrete done-condition, explicit scope boundaries, and a day-by-day learning plan — so you spend time building, not deciding what to build.
