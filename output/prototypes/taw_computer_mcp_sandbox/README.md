# taw-computer MCP Sandbox

**Give any AI agent a real Ubuntu computer.** Open-source MCP server providing 30+ tools for browser automation, desktop control, file ops, and shell execution — all sandboxed in Docker. Works with Claude Code, Cursor, and Claude Desktop.

## Headline Result

One MCP config line gives Claude full computer-use capabilities: navigate websites, fill forms, take screenshots, run shell commands, and manipulate files — all inside an isolated Ubuntu container.
