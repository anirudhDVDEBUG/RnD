# Company Execution Layer Builder

**Turn your company's knowledge docs into runnable AI playbooks.** This tool scaffolds a structured "company brain" that separates what your company _knows_ (context: SOPs, brand guides, product specs) from what it _does_ (skills: executable playbooks that reference that context). Wire them together with a single `CLAUDE.md` and any team member can run any playbook instantly.

**Headline result:** From zero to a 4-skill marketplace with validated context references in under 10 seconds — no API keys, no setup.

```
bash run.sh
```

See [HOW_TO_USE.md](HOW_TO_USE.md) for setup and Claude Code integration, and [TECH_DETAILS.md](TECH_DETAILS.md) for architecture details.

Source: [Build an Execution Layer for Your Company Brain](https://www.youtube.com/watch?v=gMm50Sy_GmQ) by Bradley Bonanno.
