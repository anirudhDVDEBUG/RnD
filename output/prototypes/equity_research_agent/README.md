# Equity Research Agent

**Turn Claude Code into a Wall Street-style equity research analyst.** This skill gives Claude 8 analysis modules — income statement, balance sheet, cash flow, ratio analysis, DCF valuation, screening, risk assessment, and catalyst identification — that combine into structured research reports with buy/hold/sell ratings and price targets.

## Headline Result

```
EQUITY RESEARCH REPORT — NVDA
Rating:        BUY
Price Target:  $195.65 (Base case, 12-month)
Upside:        44.5%
Bear/Bull:     $117.50 — $316.82
```

A single command produces a multi-section report covering financials, valuation, risks, and catalysts — adapted to beginner, intermediate, or advanced investor levels.
