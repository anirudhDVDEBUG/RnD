# Browse Anthropic Skills Marketplace

**Browse, search, and get install instructions for Agent Skills from the official Anthropic skills repository (`anthropics/skills`) — without leaving your terminal.**

This prototype bundles a catalog snapshot of 12 skills across 4 categories (Documents, Development, Creative, Enterprise) and lets you search, filter, and generate install commands for Claude Code, Claude.ai, or the API.

**Headline result:** Run `bash run.sh` and instantly see every available Anthropic skill with category, description, trigger phrases, and copy-paste install instructions.

- [HOW_TO_USE.md](HOW_TO_USE.md) — install steps, trigger phrases, first 60 seconds
- [TECH_DETAILS.md](TECH_DETAILS.md) — architecture, limitations, why it matters
