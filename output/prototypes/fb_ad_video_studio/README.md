# FB Ad Video Studio

**Generate high-converting Facebook/Instagram/TikTok video ads entirely as code** using HyperFrames HTML compositions with a battle-tested 5-part ad structure (Hook, Problem, Solution, Proof, CTA), audio pipeline, and reverse-template workflow.

## Headline Result

```
Input:  Product name + 5 ad copy lines + platform choice
Output: Interactive HTML video ad + audio timing cues + reusable template
        Ready to screen-record or render to MP4 in under 2 seconds
```
