# Ta-Persona: Digital Twin Distillation

**Imprint anyone's tone of voice and cognitive OS through guided interviews, producing a reusable persona document that makes any LLM write in their voice.**

## Headline Result

```
Input:  5-round guided interview (10 adaptive questions)
Output: Structured persona profile with expression DNA, reasoning style,
        vocabulary patterns, and calibration examples -- ready to use as
        a system prompt prefix.
```

A demo run with mock data completes in under 2 seconds and produces a full
persona Markdown + JSON file in `output/`.

## Quick Start

```bash
bash run.sh
```
