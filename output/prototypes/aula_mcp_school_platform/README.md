# Aula MCP School Platform

**MCP server for Denmark's Aula school platform — TypeScript MitID auth, no headless browser. Exposes profiles, calendar, messages, and ugeplaner (weekly plans) to AI agents via Model Context Protocol.**

## Headline Result

Connect Claude Desktop to your children's school data in Denmark — read messages, check calendars, and view weekly plans without opening the Aula app or browser.
