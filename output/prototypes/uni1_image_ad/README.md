# uni1-image-ad

**Generate Meta (Facebook/Instagram) image ads using Luma's uni-1 model. Reverse-engineer any existing ad into a reusable prompt template, then generate new creatives for any brand in seconds.**

## Headline Result

```
Input:  "Generate a 1:1 Instagram feed ad for AcmeCoffee, dark moody aesthetic, product hero shot"
Output: ad_acmecoffee_hero_1x1.png  (1024x1024, ready to upload to Meta Ads Manager)
```
